# Лабиринт — текстовый квест (Junior-версия)

Текстовая игра-квест на C++. Вы играете за персонажа, который оказался в подземном лабиринте. Цель — решить головоломки, открыть закрытые локации и выбраться на поверхность.

## Описание игры

В лабиринте **4 локации**, каждая отрисована в терминале с помощью ASCII-символов:

| Локация | Состояние | Головоломка |
|---------|-----------|-------------|
| Вход в лабиринт | открыта (старт) | — |
| Зал с колоннами | закрыта | математическая задача |
| Библиотека | закрыта | загадка |
| Выход на поверхность | закрыта | анаграмма |

Из входа можно попасть в зал или библиотеку. Каждая промежуточная локация ведёт к выходу. Попадание в открытый выход — победа.

## Сущности

- **Персонаж** — хранит текущую локацию; умеет взаимодействовать с закрытыми (головоломки) и открытыми (перемещение) локациями.
- **Локация** — ID, ASCII-форма, связи с другими локациями, состояние (открыта/закрыта), головоломка (если закрыта).
- **Головоломка** — три разных типа: математическая, загадка, анаграмма.

## Требования

- Компилятор C++17 или новее (g++, clang++ или MSVC)
- Windows / Linux / macOS

## Сборка и запуск

### Visual Studio (MSVC) — Windows

```cmd
cl /EHsc /std:c++17 /utf-8 main.cpp
chcp 65001
main.exe
```

Флаг `/utf-8` нужен для корректного отображения русского текста в консоли.

### g++ (MinGW / Linux / macOS)

```bash
g++ -std=c++17 -o labyrinth main.cpp
./labyrinth
```

На Windows:

```bash
g++ -std=c++17 -o labyrinth.exe main.cpp
labyrinth.exe
```

## Управление

После запуска доступно меню:

1. **Решить головоломку** — выбрать закрытую соседнюю локацию и ответить на вопрос.
2. **Перейти** — переместиться в открытую соседнюю локацию.
3. **Показать локацию** — повторно отрисовать текущую локацию.
0. **Выход** — завершить игру.
